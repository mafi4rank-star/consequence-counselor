# 🎯 Consequence Counselor

A digital tool designed for school counselors to help students understand the consequences of their decisions through interactive, personalized scenarios.

## 📋 Overview

**Consequence Counselor** is a web-based application that helps counselors and students explore decision-making in a safe, engaging environment.

## 🎮 Features

### For Counselors
- Simple form to input client information
- Multiple scenario templates (Substance Use, Bullying, Academic Dishonesty, Relationships, Social Media)
- Professional interface

### For Students  
- Engaging, safe space to explore decision-making
- Interactive choice scenarios with immediate feedback
- Animated heart feedback system
- Wellbeing meter showing impact of choices
- Personalized consequence messages

## 🚀 Getting Started

Just open `index.html` in your browser!

## 🎯 How It Works

1. Counselor enters client name, age, grade
2. Selects scenario type
3. Student goes through interactive scenarios making choices
4. Each choice shows consequences and impacts wellbeing
5. Personalized messages reinforce lessons

## 📝 Customization

Edit `script.js` to add new scenarios or modify messages.
